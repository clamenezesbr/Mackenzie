# UC003 – Gabriel Rezende Menezes -10419003 - Atualizar Progresso de Atividades

## Ator Principal
Devs/Designers

## Descrição
Este caso de uso descreve o processo em que Desenvolvedores ou Designers atualizam o status de suas atividades. O sistema valida as permissões e, seguindo a regra de negócio de "Include", dispara automaticamente o recálculo das métricas de progresso do projeto.

## Fluxo Principal
1. O Dev/Designer solicita a atualização de progresso de uma atividade específica.
2. A interface do sistema encaminha os dados para o controlador.
3. O controlador valida se o usuário possui permissão para editar a atividade.
4. O serviço de atividades processa a alteração e salva o novo status.
5. O sistema executa o processo de atualização de métricas (conforme regra de inclusão).
6. O serviço de métricas recalcula os indicadores e retorna a confirmação.
7. O sistema exibe uma mensagem de sucesso ao usuário.
