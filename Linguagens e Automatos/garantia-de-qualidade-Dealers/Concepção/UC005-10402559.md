# UC005 –HEITOR MACIEL DE VASCONCELLOS LEITE - 10402559 -  Analisar Previsão de Lançamento

## Ator Principal
Marketing/Publicação

## Descrição
Este caso de uso descreve o processo em que a equipe de Marketing/Publicação analisa a previsão de lançamento de projetos. O sistema coleta dados atualizados de progresso e, conforme regra de negócio de "Include", executa automaticamente a atualização de métricas para garantir maior precisão na previsão.

## Fluxo Principal
1. O usuário solicita a análise da previsão de lançamento.
2. A interface do sistema encaminha a solicitação ao controlador.
3. O controlador valida se o usuário possui permissão.
4. O sistema consulta os dados de progresso do projeto.
5. O sistema executa a atualização de métricas (conforme regra de inclusão).
6. O serviço de métricas recalcula os indicadores.
7. O controlador processa a previsão de lançamento com base nos dados atualizados.
8. O sistema exibe o resultado da análise ao usuário.
